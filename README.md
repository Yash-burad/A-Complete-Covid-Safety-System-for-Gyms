# A-Complete-Covid-Safety-System-for-Gyms
Automates management of gym during Covid

Employs mask detection system to check if members are wearing masks before entering the gym


![Mask Detection](https://user-images.githubusercontent.com/53873614/147880597-01937c7b-6789-40e4-a4f7-7c1bafc03452.jpg)

Checks if members are following social distancing norms using YOLO (displays red boxes if not) and counts the total number of members inside the gym


![Social distancing detection](https://user-images.githubusercontent.com/53873614/147880603-53172d43-a901-4af9-947c-c207a92c6f6a.jpg)

Generates an alert (top left)if a member falls during exercise by employing a fall detection mechanism


![Fall detection](https://user-images.githubusercontent.com/53873614/147880590-31849410-d5f1-4a26-b07d-e8b9193b9ae4.jpg)
